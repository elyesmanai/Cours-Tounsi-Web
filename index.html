3- HTML :

taw najmou nibdew niktbou notre code HTML. HTML howa "language de balisage", ya3ni 9oul 3lik tista3mil fil legos bich tibni karhba. il karhba tist7a9 4 3jeli, moutour, 6 bilarat, 2 rétroviseurs etc...
winti tekhou mil collection légo te3ik, 4 pièces ismhom "3ajla", 1 pièce ismha "moutour", 6 pièces ismhom " rétroviseur" etc...

fil HTML, les pièces légos te3na houma "paragraphe", "taswira", "titre", "vidéo", "liste" etc...
bich le navigateur ifari9, une fois t7il il index.html, entre ktiba 3adiya et code HTML, lezim t7ot il pièce te3ik entre "balises" ou "tags" en anglais :


taswira => <taswira>
video => <vidéo>
ghneya => <ghneya>

hedhom ismhom "self-closing" tags, w nista3mlohom fi les pièces li mafihomch contenu, exemple taswira, vidéo wala ghneya, bich t9oul li fama 7aja minhom houni, mais makch inti bich tiktib chnaw fiha il taswira. bich njiboha min blasa okhra, wataw nchoufou kifeh


titre => <titre> contenu du titre </titre>
liste => <liste> contenu de la liste </liste>
paragraphe => <paragraphe> contenu du paragraphe </paragraphe>

hedhom ismhom "normal" tags, w nista3mlohom fi les pièces li fihom contenu kima paragraphe, titre, liste d'objets, etc...
fil 7ala hedhi, lezim tiktib se3a il tag loula li ismha "opening" tag kima <paragraphe>, ba3dha tiktib chnaw fih il paragraphe edheka, w ba3d tsakir bi "closing tag" li howa nafsou kil "opening" tag ama zeyda "/", kima </paragraphe>
